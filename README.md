# Employment-Rates

A stylish, interactive web dashboard visualizing Canadian employment trends. 
Data is sourced automatically from Statistics Canada's Web Data Service API.

## Features
- Total Employment, Unemployment Rate, and Full-time / Part-time metrics
- Interactive date slicers (1-Year, 5-Year, 10-Year, All-Time)
- Month-over-Month changes and benchmark relative changes
- Automated data fetching pipeline

## How to Update
Run `update.bat` to fetch the latest data from Statistics Canada and push the updates directly to GitHub pages.
